Temp Chat Info

Image Usage

When you want to use an image in Temp Chat, you need to get the image URL. This means that when you copy and image, you need to copy the image URL and not the image itself.


Once you have gotten the link to the image, you would paste it in where you send messages. Once you have done this the link will be sent with the image. Please keep in mind that only images will work with this method and that videos will not renter properly.

Room Usage

When you want to use a room in Temp Chat you first need a good name for the room. Once you have one, you can go up to the change room button and beside it is the text input for the room. By default, the room is set to "home" but you can change it to what you like.


When you change the room you might realize that the room hasn't changed and that the text from your previous room is still there. This means that you just created a brand new room and have to be the first person to send a message for the room to update. Once you have, the room will be offically created and you can invite all your friends to it.

Unable To Send Messages

If you are unable to send messages this means you are muted. This can be identified when you press the send message button but the text doesn't clear and your message isn't sent. If you are muted, you can 1. change your username to a different one or 2. ask an admin to unmute you.

Unable To Use A Certain Username

If you are unable to use a certain username then it means that username is banned. You can tell that a username is banned when your page reloads upon entering it in. Your page also may reload when you have the username currentally, this means that it was recently banned. You can either contact an admin to unban your username or use a different one.

Unable To Enter Any Username

If you are unable to enter ant username or don't have the ability to join the chat, this means you are IP banned. This type of bans are unappealable. The IP is a public IP which means that you may be banned even though you should be. Contact an admin if you are.
